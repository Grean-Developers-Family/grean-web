# GFS 2: Web devlopment with react

## How to run 
```sh
$ npm start
```
After run this command do not need to re-compile src code. It will be done
automatically.

## Run by docker
### Docker build
```sh
$ docker build -t greanweb .
```

### Docker run
```sh
$ docker run --rm -p 6969:3000 greanweb
```
